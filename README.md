## 🚀 Featured Project

<div align="center">

### 🏭 AI-Driven Predictive Maintenance Pipeline

[![Python](https://img.shields.io/badge/Python-3.12-blue?logo=python&logoColor=white)]()
[![Scikit-Learn](https://img.shields.io/badge/Scikit--Learn-1.8-orange?logo=scikit-learn&logoColor=white)]()
[![CRISP-DM](https://img.shields.io/badge/Methodology-CRISP--DM-green)]()
[![Status](https://img.shields.io/badge/Status-Production_Ready-success)]()

**End-to-end Machine Learning system for predicting equipment failures in industrial rotary machinery.**

> 🎯 **88.2% Recall** in failure detection | 📊 **Interactive Dashboard** | 📄 **Full Technical Report**

[![View Repository](https://img.shields.io/badge/📂_View_Repository-ML_PredictiveMaintenance_afz-2ea44f?style=for-the-badge)](https://github.com/Al-Fuentes-27/ML_PredictiveMaintenance_afz)
[![Live Dashboard](https://img.shields.io/badge/🚀_Live_Demo-Interactive_Dashboard-0078D4?style=for-the-badge)](https://al-fuentes-27.github.io/ML_PredictiveMaintenance_afz/dashboard/dashboard.html)

**Key Achievements:**
- ✅ Random Forest model with **AUC-ROC=0.9710** and **Recall=0.8824**
- ✅ CRISP-DM methodology applied across 6 structured phases
- ✅ SMOTE for handling imbalanced data (3.39% failure rate)
- ✅ Interactive HTML dashboard with Plotly.js and Tailwind CSS
- ✅ Comprehensive technical report with APA 7 references

**Tech Stack:** Python • Pandas • NumPy • Scikit-Learn • Imbalanced-Learn • Plotly • Matplotlib • Seaborn
